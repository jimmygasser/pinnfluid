# Structure primitive library

STL geometry for the structures the web app can place (panels, tables, cubes,
cones, cylinders, concentrators, houses, etc.). The domain builder loads these
by type name (`single_stl/<type>.stl`).

The two wind-turbine meshes (`largewt.stl`, `smallwt.stl`, ~45 MB each) are
**excluded from this beta**: they are heavy and turbine cases are the weakest
part of the current models. They are available in the full data release.

This library grows over time. If you need a structure that is not here, email
**jimmy.gasser@epfl.ch** and it can often be added.
